# min-ui
It is my components library
## Geting started
show the docs
```
npm run serve
```
## How to use
```
npm i min-comp
```
```javascript
import {MinPlugin} from 'min-comp' 
import 'min-comp/dist/style.css'


<m-card shadow="hover"> Always </m-card>
```


## package
the main component package
```
npm run build:lib
```
the docs
```
npm run build:doc
```
